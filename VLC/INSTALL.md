# [  VLC ]

## Install VLC

*From: https://www.videolan.org/vlc/download-ubuntu.html*

Download from the Linux Snap Store

```bash
sudo snap install vlc
```


Validate the installation just verifying if VLC commands are available from the Terminal

```bash
vlc --version
```
